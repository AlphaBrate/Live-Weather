ABOUT THIS PROJECT

AlphaBrate/Live-Weather

This is a open source project built by AlphaBrate under the APEL License.

Lead Developer: ReTrn Chu

Our project draws inspiration from the Live Weather Wallpaper feature found exclusively on iOS. Recognizing its absence on other platforms, we took the initiative to bring this immersive experience to the web.

By using this project, you agree to our Terms & Conditions.

Use Your Own API Key.

As we don't charge users and can't cover high-traffic API costs, we suggest using your own API key if available. Rest assured, your key will be securely stored in your browser's local storage and won't be shared with anyone, including us.

We use a free geocoding API service which is provided by Map Maker.

Visit https://geocode.maps.co/ and tap on Free API Key. Please enter your email address. You will then receive a confirmation email. After you have confirmed your email address, they will send you your API key in a separate email. Last, enter your API key in the Geocode API Key below.

.

If you're using third-party services like Wallpaper Engine to display your desktop wallpaper, you might encounter issues with location services not functioning as anticipated. In such situations, you have the option to manually enter your location.

Visit https://live-weather-background.web.app/settings?tap=api in your system's default browser press Get Your Location, and then copy the location data into the Location field below.

HANDLING YOUR DATA

We do not retain any of your data on our server. Your location is exclusively used for the purpose of retrieving weather data.

We collect the following sensitive information from you:
Your geographical location (specified by Latitude and Longitude);
Your API Key, but only if you opt to provide one.

All third-party services we utilize operate under the Hypertext Transfer Protocol Secure (HTTPS), fortified with SSL/TLS encryption. This guarantees the security and privacy of your data during transmission over the internet.
